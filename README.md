# 📚 Library Management System (Using HTML Frames)

## 📌 Description
A simple Library Management System UI built using **HTML Frames**.
This project demonstrates basic layout design with a header, navigation menu, and content section.
---
## 🚀 Features
* Header section with project title
* Side navigation menu
* Multiple pages (Home, Add Book, View Books)
* Frame-based layout navigation
---
## 🛠️ Technologies Used
* HTML
* Frames (`<frameset>`, `<frame>`)
